# 🌤️ AI Weather Assistant

An intelligent weather application with an AI chatbot that answers questions about weather, outfits, activities, travel safety, and health.

## ✨ Features

- ✅ Real-time weather data
- ✅ AI Chat Assistant
- ✅ Smart outfit recommendations
- ✅ Health advisory
- ✅ Travel safety scores
- ✅ Activity suggestions
- ✅ Responsive design

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
- Font Awesome
- OpenWeatherMap API

## 🚀 Live Demo

[View Live Demo](https://github.com/ManasiMhaskar/ai-weather-assistant.git)


## 🔧 Installation

1. Clone the repository
2. Get API key from OpenWeatherMap
3. Add API key to config.js
4. Open index.html in browser
